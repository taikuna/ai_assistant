# AI秘書 - 要件定義書

## 概要

LINEグループ内で動作するAIアシスタント。お客様からの依頼を受け付け、業務を効率化する。

## 技術スタック

| 項目 | 選定 |
|------|------|
| AIモデル | Gemini 2.5 Flash |
| プラットフォーム | LINE（グループチャット） |
| カレンダー連携 | TODO: 要検討（Googleカレンダー？） |

## 主要機能

### 1. 依頼受付・管理

- [ ] お客様からの依頼メッセージを認識
- [ ] 依頼内容を構造化して保存
- [ ] 納期の設定・管理

- [ ] 依頼内容に不足情報がある場合、自動で質問
